Proceso de instalación de aplicación "VAMCI".
Estos pasos son realizados en dispositivo modelo Samsung SM-J610G con sistema operativo Android 10, y son escalables a otros dispositivos Android.
1. Descargar el archivo de extensión .apk correspondiente a la aplicación VAMCI.
2. Dar clic sobre el archivo. Esto lo llevará a una ventana de advertencia sobre instalación de aplicación no verificada o de fuente desconocida.
3.  Otorgar permiso de instalación sin verificación accediendo al botón "Configuración" perteneciente a la alerta y activando la opción "Permitir desde esta fuente".
4. Volver a la ventana anterior (ventana de instalación) y estará abierta una advertencia de confirmación para instalar la aplicación. A continuación se selecciona instalar y el proceso estará completo.
